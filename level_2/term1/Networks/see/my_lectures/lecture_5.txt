• *عوامل معدل البيانات Data rate :*
1) توافر الـ Bandwidth .
2) مستوى الاشارات التي نستخدمها
3) جودة القناة (مستوى الضوضاء) .

• زيادة مستويات الاشارة تزيد من احتمال حدوث اخطاء لان التباعد بين المستوى ينخفض .
——————————————————————
• *نظرية نيكويست* تنص على ان تكون القناة بلا ضوضاء وقانونه (ايجاد BitRate الذي هو Capacity ووحدته bps) 
C = 2 B log2 2*n

@ *bandwidth = 3000, signal levels = 2, find C ?*
C = 2 B log2 2*n
2×3000×log2 2² = 6000bps
——————————
@ *bandwidth = 3000, signal levels = 4, find C ?*
C = 2 B log2 2*n
   2×3000×log2 2⁴
= 2×3000×log2 4  = 12000bps

علاقة طردية بين C و B او Levels
علاقة عكسية بين الضوضاء و C
——————————————————————
• *نظرية شانون* تعطي سعةC النظام بوجود الضوضاء (الضوضاء تكون قوية جدا اذا كانت الاشاره ضعيفه)
C = B log2(1 + SNR)

@ *SNR = 0, find C ?*
C = B log2(1+0)= B log2 1
C = B×0 = 0
——————————
@ *SNR = 3162, Bandwidth = 3000, find C ?*
C = 3000 log2(1+3162)
C = 3000 log2 3163
C = 3000 × 11.62 = 34860 bps
——————————
@ *SNR db= 36, Bandwidth = 2 MHz ?*
SNR = 10³٦/¹° = 10³٠٦ = 3981
B = 2 × 10٦ = 2.000.000
C = 2.000.000 log2 3982
C = 24 bps

• علاقة طردية بين C و (B او SNR)
• لايمكن التعويض بـdb لذا نحوله لـlinear
